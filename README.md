# 🌸 Album

A beautiful sakura-themed photo book app built with pure HTML, CSS, and JavaScript. Upload your photos, apply filters, add stickers, create collages, and organise everything into custom folders — all running entirely in the browser with no dependencies or build step required.

---

## ✨ Features

- **Gallery** — Upload photos via click or drag & drop (JPG, PNG, WEBP)
- **Photo Editor** — Apply 9 filters and drag-and-drop stickers onto your photos
- **Collage Maker** — Arrange photos into 2, 3, 4-grid, or full single layouts
- **Folders** — Create custom folders with emoji icons to organise your saved photos
- **Save to Device** — Download your edited photos directly to your device
- **Sakura Theme** — Floating petal animations and a full cherry blossom colour palette throughout

```
sakura-album/

└── app.js       # All interactivity — gallery, editor, collage, folders
```

## 🚀 Getting Started

No installation or build step needed. Just open the app in your browser:

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/sakura-album.git

# Navigate into the folder
cd sakura-album

# Open in your browser
open index.html
```

Or simply double-click `index.html` on your computer.
